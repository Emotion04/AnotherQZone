# 另一个QQ空间

> 一个人的互联网痕迹 —— QQ空间**说说**与**日志**的极简黑白档案馆。

以 `moods-all.json`（说说，含文本 / 时间戳 / 图片）与 `blogs-ai/`（日志 Markdown）
为数据源，自动解析、清洗、归档，生成一个可切换 Tab、带精美时间轴与照片画廊的个人网页。

---

## 前言
或许是害怕曾经的记忆消失。毕竟，新浪博客相册、网易相册、腾讯微博等相继停运，互联网产品都有着自己的生命周期。QQ的兴趣部落、匿名说、随心说涂鸦等等相继下架关停，不赚钱就砍掉，正是很多大厂的真实写照。

于是萌生了备份QQ空间的念头，于是乎，这个项目诞生了。

## 快速开始

### 方式一：本地服务器（推荐，数据实时同步）

```bash
node serve.mjs
# 浏览器打开 http://localhost:4321
```

每次刷新页面都会**重新读取源文件**——你更新 `moods-all.json` 或 `blogs-ai/`，
刷新即同步，无需任何额外操作。指定端口：`node serve.mjs 8080`

### 方式二：静态快照（可直接双击打开）

```bash
node build.mjs          # 生成 site/data.js
# 然后直接双击 site/index.html 即可（无需服务器）
```

更新数据后重新执行 `node build.mjs` 即可刷新快照。适合归档、拷贝、部署到静态托管。

### 可选：把照片永久存档到本地

说说里的照片默认直接引用 QQ 图床链接。想做**永久离线存档**：

```bash
node download-images.mjs        # 下载全部图片到 site/media/（断点续传，已存在自动跳过）
node build.mjs                  # 重新构建；网页会自动改用本地图片
```

> 需要 Node.js，**零第三方依赖**。

### 从零抓取你自己的数据

本仓库只负责**解析与展示**，不含任何真实数据。要得到 `moods-all.json` / `blogs-ai/`，
用配套的导出器抓取你**本人**的 QQ空间（详见 [`exporter/README.md`](exporter/README.md)）：

```bash
node exporter/qzone-export.js <你的QQ号>
```

它输出的字段与网站期望**天然一致**，因此格式不会对不上。若不用它、改用别的爬虫，
只要把输出对齐「数据格式约定」即可，否则 `node build.mjs` 会打印告警。

> 🔒 **隐私**：你的说说、日志、图片以及**他人的评论/访客姓名**都在 `.gitignore` 中，绝不入库。
> 仓库里自带一份 `*.sample` 假数据 —— clone 下来直接 `node build.mjs` 就能看演示。

## 功能

- **四个 Tab**
  - `概览` —— 数字统计、年度活跃轨迹图、日志分类
  - `时间轴` —— 说说与日志按时间倒序交织的精美竖向时间轴（本馆核心组件）
  - `说说` —— 全部说说按年分组，含**照片画廊**、精确时间、多段长文
  - `日志` —— 卡片列表，点击进入沉浸式阅读浮层（含原文评论、访客）
- **照片画廊 + 灯箱** —— 说说缩略图懒加载，点击全屏看大图，多图可左右翻（← → / 点击）
- **简洁模式** —— 顶栏一键切换，**自定义**是否展示赞 / 评论 / 访客，记忆偏好
- **极简黑白** —— 纯单色系统，衬线标题 + 无衬线正文，发丝线与大量留白
- **明暗双主题** —— 右上角一键切换，记忆偏好
- **全文搜索** —— 实时高亮命中，跨说说 / 日志
- **年份筛选** —— 时间轴与说说均可按年过滤
- **URL 深链** —— `#timeline` `#moods` `#blogs` 可收藏 / 分享具体 Tab
- **响应式** —— 手机、平板、桌面自适应

## 目录结构

```
Qzone/
├── moods-all.json        说说数据源（结构化：文本 / 时间戳 / 图片URL）· 私有，不入库
├── blogs-ai/             日志数据源（每篇一个 .md）· 私有，不入库
├── moods-all.sample.json 示例假数据（供演示 / 说明格式）
├── blogs-ai.sample/      示例假日志（含示例评论 / 访客）
├── parse.mjs             零依赖解析器：解析 + 清洗 + 统计 + 告警
├── build.mjs             生成静态快照 site/data.js
├── serve.mjs             本地服务器，每次请求实时解析
├── download-images.mjs   把图片下载到 site/media 做永久存档
├── exporter/             QQ空间导出器（抓取逻辑 + 说明）
├── package.json          npm run build / serve
└── site/
    ├── index.html        页面结构
    ├── styles.css        黑白样式系统
    ├── app.js            视图渲染 / 时间轴 / 画廊灯箱 / 搜索 / 阅读浮层
    ├── media/            本地图片存档（运行下载脚本后生成）· 不入库
    └── data.js           自动生成（勿手改）· 不入库
```

## 数据格式约定

新增数据保持同样格式即可被自动收录。若格式变化，`node build.mjs` 会打印**告警**提醒。

**说说** `moods-all.json` —— 数组，每条：
```json
{ "c": "文本（可含换行）", "t": "14:24", "ts": 1783664647, "u": ["图片URL", ...] }
```
`c` 里的 QQ 表情码 `[em]eXXX[/em]` 会被自动清除；时间以 `ts`（Unix 秒，北京时区）为准。
若 `moods-all.json` 缺失，解析器会回退到 `moods-ai/` 的 Markdown。

**日志** `blogs-ai/YYYY-M-D HH-MM 标题.md`
```markdown
---
type: qzone-blog
date: 2025-3-26 18:41
---

# 标题

正文……
```
QQ空间导出的样板文字（赞 / 转载 / 签名档等）会被自动剔除；
`分类 | 公开 | 原创：X` 一行解析为分类与作者；楼层**评论**与**访客**会被尽力还原
（在简洁模式下可隐藏）。
